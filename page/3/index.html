raw_input 和 input

raw_input 会把用户输入转换为字符串对象，而 input 则会保留原来的输入数据类型；然而在 python3 input 函数已经等同于 raw_input 的作用，如果想要实现 python2 input 的作用则需在 raw_input 之后使用 eval 函数将数据进行转换。
